## Core Capabilities

<Accordion title="Complete MOQT Protocol Implementation">
  Moxygen implements the full MOQT (Media over QUIC Transport) specification with support for SUBSCRIBE, FETCH, PUBLISH_NAMESPACE, and all control plane operations. Version negotiation and protocol extensions are built-in.
</Accordion>

<Accordion title="Flexible Component Architecture">
  The symmetric Publisher/Subscriber design enables powerful composition patterns. Build filters, relays, caches, and transcoders by implementing consumer interfaces to receive data and publisher interfaces to serve data downstream.
</Accordion>

<Accordion title="Production-Ready Relay">
  MoQRelay provides namespace routing, subscription management, and optional object caching. MoQForwarder handles fan-out data forwarding with per-subscriber filtering and backpressure control.
</Accordion>

<Accordion title="Real-World Media Support">
  Includes FLV parser, MoqMi media packaging, and sample applications for streaming video/audio over MOQT. Integrate with ffmpeg for broadcast-quality live streaming.
</Accordion>

## Quick Example

```cpp example.cpp
#include <moxygen/MoQClient.h>
#include <moxygen/MoQSession.h>

// Create a MOQT client and connect
auto client = std::make_shared<MoQClient>(evb);
auto session = co_await client->connect(
  proxygen::URL("https://relay.example.com:4433/moq"),
  settings
);

// Set up the session
co_await session->setup(/* role */ moxygen::Role::PUBLISHER);

// Subscribe to a track
auto subscribeResult = co_await session->subscribe(
  SubscribeRequest{
    .requestID = 1,
    .trackNamespace = "live-stream",
    .trackName = "video",
    .startLocation = AbsoluteLocation{0, 0}
  },
  consumer
);
```
