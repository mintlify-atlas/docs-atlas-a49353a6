## Overview

`MoQTypes.h` defines all core types, structures, and enums used in the MOQT protocol. This includes request/response messages, error codes, track identifiers, and protocol constants.

**Source**: `moxygen/MoQTypes.h`

## Core Type Aliases

```cpp
using Payload = std::unique_ptr<folly::IOBuf>;
using Priority = uint8_t;
```

## Track Identifiers

### TrackNamespace

```cpp
using TrackNamespace = std::vector<std::string>;

// Example:
TrackNamespace ns{"live-streams", "sports", "soccer"};
```

Hierarchical namespace for organizing tracks.

### FullTrackName

```cpp
struct FullTrackName {
  TrackNamespace trackNamespace;
  std::string trackName;
  
  struct hash {
    size_t operator()(const FullTrackName& ftn) const;
  };
};

// Example:
FullTrackName ftn{
  .trackNamespace = {"live-stream"},
  .trackName = "video"
};
```

### TrackAlias

```cpp
using TrackAlias = uint64_t;
```

Numeric alias for a track (set by publisher, used in object headers).

## Location Types

### AbsoluteLocation

```cpp
struct AbsoluteLocation {
  uint64_t group;
  uint64_t object;
  
  bool operator<(const AbsoluteLocation& other) const;
  bool operator==(const AbsoluteLocation& other) const;
};
```

Identifies a specific object by group and object ID.

### SubscribeRange

```cpp
struct SubscribeRange {
  std::variant<
    AbsoluteLocation,    // Start from specific object
    LatestGroup,         // Start from latest group
    LatestObject         // Start from latest object
  > start;
  
  std::optional<AbsoluteLocation> end;  // Optional end location
};
```

Defines the range of objects for a subscription.

## Object Headers

### ObjectHeader

```cpp
struct ObjectHeader {
  uint64_t group;
  uint64_t object;
  uint64_t length;                  // Payload length
  Priority priority;
  ForwardingPreference forwardingPreference;  // Track, Subgroup, Datagram
  ObjectStatus status;              // Normal, EndOfGroup, EndOfTrackAndGroup
};
```

## Usage Examples

### Creating a Subscribe Request

```cpp subscribe_example.cpp
SubscribeRequest subReq{
  .requestID = 1,
  .trackAlias = 0,  // Will be set by session
  .trackNamespace = {"live-stream"},
  .trackName = "video",
  .range = {
    .start = LatestGroup{},
    .end = std::nullopt  // Subscribe to end
  },
  .filter = FilterType::LatestGroup,
  .groupOrder = GroupOrder::OldestFirst,
  .parameters = {}
};
```

### Creating a Fetch Request

```cpp fetch_example.cpp
Fetch fetchReq{
  .requestID = 2,
  .trackAlias = 0,
  .trackNamespace = {"vod"},
  .trackName = "video",
  .start = AbsoluteLocation{/* group */ 0, /* object */ 0},
  .end = AbsoluteLocation{/* group */ 100, /* object */ 0},
  .groupOrder = GroupOrder::OldestFirst,
  .parameters = {}
};
```

### Working with Extensions

```cpp extensions_example.cpp
Extensions extensions{
  {kDeliveryTimeoutExtensionType, encodeVarInt(3000)},  // 3 second timeout
  {kPublisherPriorityExtensionType, encodeVarInt(128)},  // Default priority
  {kPublisherGroupOrderExtensionType, encodeVarInt(GroupOrder::OldestFirst)}
};
```
